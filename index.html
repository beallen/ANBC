2018 Monitoring Program

The Alberta Native Bee Council (ANBC) has implemented a province-wide sampling program for wild bees. This program samples native bees in 2018 using passive sampling techniques (blue vane traps) that can help inform the status and distribution of this group. Depending on the quality/quantity of the species data, we plan to address the following objectives:
  1. Create range maps for all species.
  2. Describe the relative abundance of each species.
  3. Assess the landscape-level impacts on native bee populations using species distribution models and/or use availability models.
  4. Identify how communities of bee species are distributed across the province.
The ANBC is collaborating with the Alberta Biodiversity Monitoring Institute (ABMI) to analyze the ~10,000 unique species collected by the ANBC.
